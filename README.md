# Amazon Clone 🛒

A responsive front-end Amazon Clone built using **HTML5** and **CSS3**. This project recreates the basic user interface of Amazon's homepage, including the navigation bar, search functionality, hero section, product categories, and footer.

## 🚀 Features

* Responsive Navigation Bar
* Amazon-style Search Box
* Location Delivery Section
* Sign In & Orders Section
* Shopping Cart Icon
* Hero Banner Section
* Product Category Cards
* Multi-Column Footer
* Clean and Modern UI Design
* Font Awesome Icons Integration

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome
